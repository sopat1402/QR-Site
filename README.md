# QR-Site
A simple QR code generator with both frontend and backend functionality that can encode URLs, Plain Text, Phone Numbers and SMS.

There are a few instructions for use:
1) index.html, styles.css and script.js are the FRONT END files
2) index.js, package-lock.json and package.json are the BACKEND files.
3) To start the server, do => "node index.js" in a terminal in that path
4) But before that you need to install the dependencies with => "npm i express qrcode"

I couldn't upload all the dependencies so the terminal command is in step 4 to install them.

YOU HAVE TO CHANGE THE IP ADDRESS OF THE SERVER TO USE IT WITH DIFFERENT DEVICES as it is currently set to localhost.
